# 🚨 Risk Dependency Mapper

## 📌 Project Overview
Risk Dependency Mapper is a full-stack web application developed using React JS, Spring Boot, and MySQL.  

The system helps organizations manage and track risks efficiently with CRUD operations and severity-based classification.

---

## 🚀 Features
- Add Risk
- Edit Risk
- Delete Risk
- View Risk Dashboard
- Severity Color Coding
- REST API Integration
- MySQL Database Storage
